## Getting Started

### 1- Clone this repo:

**git clone https://gitlab.com/contact.aftercode/erpygo_documents**

### 2- Download pnpm :

**npm install -g pnpm**

### 3- Install the dependencies of the project:

**pnpm i**

### 4- Run the local server:

**pnpm run dev**
